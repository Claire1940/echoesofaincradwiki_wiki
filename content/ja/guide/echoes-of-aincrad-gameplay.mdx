### スタミナとSPのリソース管理

最前線で勝利を収めるには、スタミナとSPという2つの主要な戦闘リソースのバランスを取る必要があります。ダッシュ、回避、盾によるガードなどの標準的なアクションはスタミナゲージを消費します。武器固有の強力なソードスキルを繰り出すにはSPを消費します。重攻撃に頼りすぎると完全に無防備な状態になり、ボタン連打は敗北への近道となります。

### 防御オプションとカウンターアタック

Echoes of Aincrad は、その深い防御マニューバによって標準的なハックアンドスラッシュゲームとは一線を画しています。装備している装備に応じて、3つの主要な防御ツールにアクセスできます。

| 防御タイプ | 入力条件 | 最適な使用場面 | 戦術的報酬 |
| :--- | :--- | :--- | :--- |
| **ジャスト回避** | 回避ロールの正確なタイミング | 回避不能なボスの重攻撃 | 一時的な無敵時間の付与 |
| **シールドガード** | 盾を構えてガードボタンを保持 | 継続的な物理攻撃 | 直接的なHPダメージの最小化 |
| **パリィ / リバーサル | ガードのシビアなタイミング | 標準的な敵の武器攻撃 | 敵の体勢を崩しパートナーコンボを誘発 |

<Section icon={Shield} title="デスゲームモード：ハイリスクな生存ルール">

究極のソードアート・オンライン体験を求めるプレイヤーのために、Echoes of Aincrad にはオプションの**デスゲームモード**が用意されています。このモードは、アニメの原作設定である生か死かの賭けを直接再現したものです。

```
+-------------------------------------------------------------+
|                    デスゲームモード・ルール                   |
+-------------------------------------------------------------+
|  [ プレイヤーの敗北 ] ------> [ キャラクターデータ削除 ]       |
|                                                             |
|  [ 技術的な中断 ] ----------> [ セーブデータは安全に保持 ]     |
|                                                             |
|  [ デモ版の進行 ] ----------> [ デスゲームモードへ変換不可 ]   |
+-------------------------------------------------------------+
```

### 主要な生存ルール

- **永久的な敗北**: このモードでキャラクターのHPがゼロになると、そのキャラクターのセーブファイルは永久に削除されます。
- **リスポーンなし**: セカンドチャンス、蘇生アイテム、チェックポイントへのロールバックは一切存在しません。
- **安全な切断**: ゲームを通常通り終了したり、メインメニューに戻ったり、突然の停電が発生したりしても、セーブデータは削除されません。ペナルティはゲーム内での実際の敗北時にのみ発生します。
- **解放条件**: 通常版のプレイヤーはメインストーリーをクリアした後にこのモードが解放されます。デラックスおよびアルティメットエディションの所有者は、最初のプレイからこのリスクに挑める**早期解放**特典を受け取れます。

</Section>

### デモ版セーブデータの引き継ぎ方法

1. **セーブデータを保持**: コンソールやPCのストレージからデモ版のセーブファイルを削除しないでください。
2. **製品版をインストール**: 同じプラットフォームで Echoes of Aincrad の製品版をダウンロードし、インストールします。
3. **パッチの適用**: ゲームがネットワークに接続され、最新バージョンにアップデートされていることを確認します。
4. **プロンプトに従う**: 製品版を起動し、初期規約を読み、システムがデモデータを検出した際に「引き継ぐ（Carry over）」を選択します。
5. **ゲーム開始**: 決定ボタンを長押しして引き継ぎ画面を閉じ、メインメニューから「Game Start」を選択します。
