<Section icon={Swords} title="武器クラスとソードスキル">

アインクラッドの世界では、あなたのアイデンティティは手にする鋼によって定義されます。体験版には6つの異なる武器タイプが登場し、それぞれが独自のプレイスタイルと解放可能な**ソードスキル**を持っています。従来のRPGとは異なり、キャラクター作成時にクラスを選択することはありません。代わりに、特定の武器で敵を倒すことで武器経験値を獲得します。ソードスキルは最大3つまで装備可能で、戦闘スタイルを大幅にカスタマイズできます。

| 武器タイプ | 戦闘スタイル | 防御手段 | 主な利点 |
| :--- | :--- | :--- | :--- |
| **片手剣と盾** | バランス型 | シールドブロック | 初心者やソロプレイに汎用性が高い |
| **両手斧** | 攻撃型 | なし | 高い範囲ダメージとクラウドコントロール |
| **短剣** | 高速/俊敏 | 遠距離投擲 | 連続ヒットと高い機動力 |
| **細剣** | 優雅 | 盾装備可能 | チェイン攻撃やカウンターに最適 |
| **両手剣** | 重突進型 | 回避ロール | 強力な溜めダメージ出力 |
| **メイス** | 打撃型 | 盾装備可能 | 高い怯み性能とガード崩し |

これらの武器を使いこなすには、スタミナ（SP）消費を理解する必要があります。弱攻撃からパリィに至るまで、あらゆるアクションがSPを消費します。このリソース管理が、クエストの成功か、ダンジョンからの無念の撤退かを分けることになります。「はじまりの街」に到着したら、鍛冶屋を訪れて武器のパラメータを強化しましょう。これは熟練度レベルとは別の強化要素です。

<Callout type="info" title="武器の管理">
装備している武器を変更できるのは、はじまりの街の宿屋にいる間だけです。新しいクエストに出発する前に正しい装備セットを選択しているか確認してください。ミッションが完了するまでその選択は固定されます。
</Callout>

</Section>

<Section icon={Target} title="キャラクターの成長とカーディナルランク">

**Echoes of Aincrad 体験版クエスト**における成長は、従来のレベリングと独自の「カーディナルランク」システムの2つの軸で進みます。キャラクターレベルはステータス割り当て用の「成長ポイント」を付与しますが、カーディナルランクはあなたの熟練度を示す世界共通の指標となります。このランクは、商人から購入できるアイテムの質や、フィールドで遭遇するモンスターの難易度を決定します。

<Steps>
  <Step num="1" title="成長ポイントの獲得">
    ダンジョン内のモンスターを倒してキャラクターレベルを上げ、成長ポイントを受け取ります。
  </Step>
  <Step num="2" title="宿屋でのステータス割り当て">
    宿屋の自室に戻り、「筋力」「器用」「敏捷」「知力」「生命力」「持久力」「精神」にポイントを振り分けます。
  </Step>
  <Step num="3" title="カーディナルランクの上昇">
    メインクエストの目標を達成したり、新しい街を発見したりしてランクを上げ、より優れた鍛冶屋のレシピやショップアイテムを解放します。
  </Step>
  <Step num="4" title="鍛冶屋での強化">
    クエスト中に集めた素材を使用して武器のパラメータをレベルアップさせ、EX-Modを組み合わせて性能を向上させます。
  </Step>
</Steps>

| ステータス名 | 主な効果 | おすすめのビルド |
| :--- | :--- | :--- |
| **筋力 (Strength)** | 物理ダメージの基礎値を増加 | 両手剣 / 両手斧 |
| **敏捷 (Agility)** | 攻撃速度と回避率を向上 | 短剣 / 細剣 |
| **器用 (Dexterity)** | クリティカル率と命中率を強化 | 細剣 / 弓（将来実装） |
| **生命力 (Vitality)** | 最大HPを増加 | 全ビルド（タンク重視） |
| **持久力 (Endurance)** | 最大スタミナ（SP）を増加 | 重武器ユーザー |

<Callout type="warning" title="難易度のスケーリング">
カーディナルランクが上がると、モンスターは大幅に手強くなります。キャラクターレベルが高くても、鍛冶屋で定期的に武器をアップグレードしていないと、パワー不足を感じることになるでしょう。
</Callout>

</Section>

<Section icon={Shield} title="戦闘戦術とパートナーとの連携">

体験版の最終クエストで生き残るには、パートナーの活用が不可欠です。各AIコンパニオンは独自の戦闘スタイルと、アクションホイールから発動できる「コンビネーションスキル」を持っています。パートナーへの指示の出し方を理解することは、大勢の敵を捌いたり、脅威度の高い単体のボスを集中攻撃したりする上で極めて重要です。

<CardGrid cols={2}>
  <Card title="スイッチモード (Switch Mode)">
    - **焦点**: 防御的な囮役。
    - **効果**: パートナーが敵のヘイト（ターゲット）を集めます。
    - **最適な使用場面**: 強力な攻撃を溜める時間や、回復する余裕が必要なとき。
  </Card>
  <Card title="フリーモード (Free Mode)">
    - **焦点**: 攻撃的な連携。
    - **効果**: パートナーが独立して攻撃するか、あなたのターゲットを集中攻撃します。
    - **最適な使用場面**: 低レベルの雑魚敵の群れを効率よく一掃するとき。
  </Card>
</CardGrid>

戦闘はSP管理の駆け引きです。新しい敵が現れるたびに、その攻撃範囲とアビリティのパターンを学習しなければなりません。例えば、体験版の蜘蛛型の敵は天井から奇襲を仕掛けてくることがよくあります。メイスで怯ませるか、短剣で遠距離から攻撃するか、どちらも有効な戦略ですが、チームの「スイッチ」マニューバとの連携が前提となります。

<Callout type="success" title="コンビネーションスキル">
パートナーのコンビネーションスキルをクイックアクセスホイールに登録しておきましょう。これらの技は甚大なダメージを与えるだけでなく、敵のアルティメットアタックを中断させることができ、反撃の重要なチャンスを作り出します。
</Callout>

</Section>
