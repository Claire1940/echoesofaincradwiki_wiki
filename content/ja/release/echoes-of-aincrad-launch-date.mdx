<Section icon={Target} title="デスゲームモードとサバイバルルール">

**Echoes of Aincradの発売日**から利用できる最もユニークな機能の一つが**「デスゲームモード」**です。このオプションのサバイバルモードは、原作『ソードアート・オンライン』の緊迫した環境を再現するように設計されています。このモードでは、ゲーム内でキャラクターが死亡すると、セーブデータが永久に削除されます。

通常、このモードはメインストーリーをクリアした後に解放されますが、デラックスおよびアルティメットエディションの所有者は、最初からアクセスすることが可能です。

| ルールのカテゴリー | デスゲームモードの仕様 |
| :--- | :--- |
| **パーマデス** | ゲームオーバー時にセーブデータが削除されます。 |
| **難易度** | 任意の難易度設定と組み合わせることができます。 |
| **セーブの安全性** | タイトル画面への戻りや停電による終了ではデータは削除されません。 |
| **解放方法** | 本編クリア、またはデラックス/アルティメット版の早期解放を利用。 |
| **体験版からの引き継ぎ** | 体験版のセーブデータをデスゲームモードに変換することはできません。 |

<Callout type="warning" title="ハイリスク警告">
デスゲームモードは上級者向けのモードです。1ライフでのランに挑戦する前に、まずは標準モードで数層をクリアし、戦闘メカニクスを習得することをお勧めします。
</Callout>

セーブデータが失われるリスクがあるため、武器のビルド、パートナーとのシナジー、アイテムの準備といったあらゆる選択が格段に重要になります。敵のヘイト管理や危機的な状況での回復など、イオリやアルゴといったパートナーとの連携を極めることが生存の鍵となります。

</Section>
