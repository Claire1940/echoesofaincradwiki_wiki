<Section icon={Swords} title="How to Transfer Demo Save Data">

One of the most frequent questions following the **echoes of aincrad demo release date** is how to carry over progress. Bandai Namco has implemented a straightforward transfer process, but it requires the full game to be updated to the latest version before the "Carry Over" prompt will appear.

<Steps>
  <Step num="1" title="Install the Full Game">
    Download and install the full version of Echoes of Aincrad on the same device where you played the demo.
  </Step>
  <Step num="2" title="Check for Updates">
    Ensure the game is updated to the latest version. Connect to the network to trigger any Day One patches.
  </Step>
  <Step num="3" title="Launch and Agree">
    Start the game and accept the end-user license agreements. The game will automatically scan for demo data.
  </Step>
  <Step num="4" title="Confirm the Prompt">
    When the "Carry over demo save data?" prompt appears, select **Yes**.
  </Step>
  <Step num="5" title="Finalize Transfer">
    Press and hold the **Confirm** button on the transfer screen. Do not turn off your console during this process.
  </Step>
</Steps>

<Callout type="success" title="Transfer Rewards">
Players who successfully transfer their data will retain all items, Col (currency), and weapon proficiency levels earned during the five demo missions.
</Callout>

It is important to note that **Death Game Mode** cannot be enabled for a save file that originated in the demo. If you wish to experience the permadeath challenge, you must start a completely new save file in the full version of the game.

</Section>
