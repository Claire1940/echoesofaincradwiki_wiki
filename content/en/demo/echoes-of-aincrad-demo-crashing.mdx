<Section icon={Settings} title="Step-by-Step Troubleshooting for PC">

If you are experiencing a crash-to-desktop (CTD) without an error message, the first step is to validate the integrity of the local game files. This process identifies corrupted assets that may have been damaged during the download process on Steam.

<Steps>
  <Step num="1" title="Verify Steam Game Files">
    Open your Steam Library, right-click on Echoes of Aincrad Demo, and select Properties. Navigate to the Installed Files tab and click "Verify integrity of demo files." This will replace any missing DLLs or corrupted textures.
  </Step>
  <Step num="2" title="Update Graphics Drivers">
    Download the latest drivers from the official NVIDIA or AMD websites. The July 2026 updates include specific optimizations for Bandai Namco's latest JRPG engine, which significantly reduces shader compilation stutters.
  </Step>
  <Step num="3" title="Clear Shader Cache">
    Navigate to your GPU control panel and clear the shader cache. For NVIDIA users, this is found under Manage 3D Settings. This forces the game to rebuild the cache, often fixing crashes that occur during the opening cinematic.
  </Step>
  <Step num="4" title="Configure Power Settings">
    Set your Windows Power Plan to "High Performance." This prevents the CPU from downclocking during intensive combat sequences, which can trigger a system-level freeze.
  </Step>
</Steps>

<Callout type="tip" title="Admin Privileges">
Right-click the game's executable in the Steam folder and select "Run as Administrator." This ensures the game has the necessary permissions to write save data and access system resources without interference.
</Callout>

</Section>

<Section icon={AlertTriangle} title="Save Data and Transfer Stability">

The demo allows players to carry over their progress to the full version of Echoes of Aincrad. However, if the demo crashes during a save operation, the resulting file may become corrupted, preventing both the demo and the full game from launching correctly.

| Save Issue | Symptom | Resolution |
| :--- | :--- | :--- |
| **Corrupted Slot** | Infinite loading on "Game Start" | Delete local save and sync from Cloud |
| **Transfer Error** | Full game fails to detect demo data | Update full game to latest July 2026 version |
| **Permission Denied** | Save failed message in-game | Disable Windows Controlled Folder Access |

<Steps>
  <Step num="1" title="Backup Local Saves">
    Locate your save folder in %LocalAppData%/BandaiNamco/EchoesOfAincrad/Saved. Copy this folder to a safe location before attempting any file transfers.
  </Step>
  <Step num="2" title="Verify Version Match">
    Ensure your demo and full game are on the same regional version (e.g., North America). Cross-region save transfers are not supported and will result in a crash.
  </Step>
  <Step num="3" title="Cloud Sync Check">
    Confirm that Steam Cloud or PlayStation Plus sync is complete before switching devices. Incomplete syncs are a primary cause of the **echoes of aincrad demo crashing** during the "Checking Save Data" screen.
  </Step>
</Steps>

<Callout type="warning" title="Death Game Mode Limitation">
Save data carried over from the demo cannot be switched to Death Game Mode. If you intend to play the one-life challenge, you must start a fresh save file rather than using transferred data.
</Callout>

</Section>
