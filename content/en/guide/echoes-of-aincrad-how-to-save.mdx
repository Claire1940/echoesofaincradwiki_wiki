<Section icon={Swords} title="Death Game Mode: Save Deletion Rules">

The **Death Game Mode** is the ultimate challenge in *Echoes of Aincrad*, simulating the high stakes of the original SAO anime. In this mode, the way you save doesn't change, but the consequences of failure are absolute. This mode is available after clearing the game once, or immediately for owners of the **Deluxe and Ultimate Editions**.

| Feature | Normal Mode | Death Game Mode |
| :--- | :--- | :--- |
| **Defeat Consequence** | Return to last checkpoint | **Character Save Erased** |
| **Auto-Save** | Enabled | Enabled |
| **Manual Save** | Enabled | Enabled |
| **Early Unlock** | No | Yes (Deluxe/Ultimate) |
| **Save Slots** | Multiple | Single Slot Only |

**How Save Erasure Works:**
If your Health Points reach zero in Death Game Mode, the game immediately triggers a "Game Over" sequence. During this sequence, the game performs a final auto-save that marks the character as "Deceased," effectively erasing the playable data. You will be returned to the title screen, and that specific save slot will be reset to empty.

<Callout type="success" title="Safe Exits">
Your save data is **safe** if you exit the game normally through the menu, or if you experience a technical interruption like a power loss or crash. The erasure system only triggers upon an **in-game defeat** event.
</Callout>

With *Echoes of Aincrad* launching across PS5, Xbox Series X|S, and Steam, managing your files across different hardware requires specific knowledge of each platform's cloud system.

**Platform-Specific Saving Details:**
- **PlayStation 5**: Uses PS Plus Cloud Storage. Ensure "Auto-Sync" is enabled in system settings to prevent data loss if your console is replaced.

- **Steam (PC)**: Supports **Steam Cloud**. This is particularly useful for players moving between a desktop PC and a **Steam Deck** (though official Deck support is still pending optimization).
