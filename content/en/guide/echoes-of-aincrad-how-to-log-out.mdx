Death Game Mode introduces a high-stakes environment where the game's save logic changes significantly. Because this mode is designed to simulate the tension of the original anime, the penalty for failure is severe.

| Feature | Standard Mode | Death Game Mode |
| :--- | :--- | :--- |
| Save Deletion | Never | Upon Defeat |
| Exit Safety | High | High |
| Progress Carryover | Supported | Restricted |

<Callout type="warning" title="Important Warning">

</Callout>
